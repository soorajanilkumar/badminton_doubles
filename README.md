# Badminton Doubles Webapp Usage

Open `doubles_webapp.html` in your browser.  
1. Enter player names and skill levels.  
2. Set the number of games and courts.  
3. Click "Generate Doubles".  
4. Select a court, assign teams, and manage games.  
5. Use "Print Games" to generate a printable schedule.
